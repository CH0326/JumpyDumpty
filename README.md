# JumpyDumpty  

## 蹦蹦炸弹——原神小工具箱

### Electron + Vue + AntDesign

## 功能：  
目前只完成的资源地图功能

    默认加载米游社的地图，游戏中的标点可同步
    支持独立地图的热键呼出和隐藏

    呼出地图窗口后，由于原神从游戏切换其他窗口默认会最小化，隐藏地图窗口时需调用windows库函数重新激活原神窗口，从而实现近似于游戏地图的体验
